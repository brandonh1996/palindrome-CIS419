RunServer.bat - Starts Server Interface
RunClient.bat - Starts Client Interface

You must run the server before starting the client. Do NOT close the Command Prompt window
while the server is running or else the client will fail to connect and the program will not run.

NOTE: These launchers will not work on SUNY - The College at Brockport computers due to firewall
restrictions. Please refer to the program's User Manual for more information.
